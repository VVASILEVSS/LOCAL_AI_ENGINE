# exchange/ — папка для обмена файлами и сообщениями

**Назначение:** общая зона между оператором (VVASILEVSS) и AI-агентами (Super Z, Hermes и др.)
для передачи артефактов, инструкций и отчётов без внешних каналов.

## Структура

```
exchange/
├── README.md              ← этот файл (описание + правила)
├── inbox/                 ← ВХОДЯЩИЕ: сюда кладёт ОПЕРАТОР для агентов
│   └── .gitkeep
└── outbox/                ← ИСХОДЯЩИЕ: сюда кладут АГЕНТЫ для оператора
    └── .gitkeep
```

## Правила использования

1. **Оператор → Агенты:** файлы в `exchange/inbox/` — коммит + push на main.
   Агент видит при следующем запуске, читает и отрабатывает.

2. **Агенты → Оператор:** файлы в `exchange/outbox/` — коммит + push на main.
   Оператор видит при pull.

3. **Именование файлов:** `YYYY-MM-DD_описание.ext`
   Пример: `2026-07-12_отчёт-по-P1-багам.md`

4. **Формат сообщений:** Markdown (`.md`) для текста, любой формат для данных.
   Каждое сообщение — отдельный файл, не перезаписывать чужие.

5. **Архивация:** старые файлы (старше 14 дней) можно удалять
   или переносить в `exchange/archive/` по договорённости.

## Для AI-агентов

При запуске проверяй:
- `exchange/inbox/` — есть ли новые задачи/инструкции от оператора
- `exchange/outbox/` — не оставил ли ты незавершённые артефакты

Сообщения от оператора всегда в `inbox/`, твои ответы — в `outbox/`.
