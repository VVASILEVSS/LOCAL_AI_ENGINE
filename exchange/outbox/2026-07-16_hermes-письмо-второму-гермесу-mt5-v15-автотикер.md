## ✅ Что сделано за сессию (MT5 индикатор v1.11 → v1.15)

### v1.11 — базовый фикс (3 бага)
1. **TF uppercase** — `ShowTFs = "15M,1H,4H,1D"` (API ждёт uppercase, был lowercase)
2. **ExtractDoubleFromPos startPos** — парс от `pos + len(key)`, не от `pos`
3. **Block truncation** — конец блока по `}` с brace counting (`depth++`/`depth--`)

### v1.12-v1.13 — разворачиваемая инфо-панель + автоцвет текста
4. **Бейдж → разворачиваемая панель** — клик по заголовку в углу разворачивает/скрывает таблицу зон
5. **Авто-цвет текста** — `GetTextColor()` через `ChartGetInteger(CHART_COLOR_BACKGROUND)`:
   - тёмный фон → белый текст (clrWhite)
   - светлый фон → тёмный текст (clrBlack)
6. **Анти-спам WR_FAIL** — `g_lastErrorKey` НЕ ресетится на файловом fallback. `WR_FAIL err=4006` пишется один раз (poll #1)

### v1.14 — multi-symbol fallback
7. **Убран `FilePath` input** — `TryReadFile(TargetSymbol)` генерирует имя `signals_<SYM>.json` динамически
8. **`FILE_SHARE_READ | FILE_SHARE_WRITE`** — фикс `err=5004` (файл залочен Flask во время записи)
9. **ETHUSDT добавлен** в `DEFAULT_AUTOSCAN_SYMBOLS` (был только BTCUSDT + XAUTUSDT)

### v1.15 — автоопределение тикера
10. **`TargetSymbol = "AUTO"`** — `ResolveSymbol()` читает `Symbol()` графика и маппит:

| График MT5 | → Файл | → Символ бота |
|------------|--------|---------------|
| `BTCUSD` | `signals_BTCUSDT.json` | BTCUSDT |
| `ETHUSD` | `signals_ETHUSDT.json` | ETHUSDT |
| `XAUUSD` | `signals_XAUTUSDT.json` | XAUTUSDT |
| `XAGUSD` | `signals_XAGUSDT.json` | XAGUSDT |
| `SOLUSD` | `signals_SOLUSDT.json` | SOLUSDT |

**Логика:** `StringToUpper(Symbol())` → `StringFind` по префиксу (BTC/ETH/XAU/XAG/SOL) → fallback `Symbol() + "USDT"`. Ручной режим: `TargetSymbol = "BTCUSDT"`.
