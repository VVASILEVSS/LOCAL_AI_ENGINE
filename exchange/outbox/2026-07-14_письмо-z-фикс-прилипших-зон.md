## Мой фикс (2 изменения)

### 1. Промпт — правило 8b (ollama_client.py:317)

```
8b. НЕ копируй одну зону на все ТФ. D1, H4, H1, M15 — РАЗНЫЕ зоны
с РАЗНЫММ lower/upper. Старший ТФ шире, младший уже. D1 ≠ H4 ≠ H1 ≠ M15.
```

### 2. Пост-обработка — _enforce_zone_uniqueness (ollama_client.py:948-994)

Если соседние ТФ идентичны (diff < 0.5% от цены) — parent расширяется outward:

| ТФ | Расширение |
|----|-----------|
| D1 | ±2.5% |
| H4 | ±1.5% |
| H1 | ±0.75% |
| M15 | 0 (не расширяется) |
| 5M | 0 |

Логика: child уже сужен в _validate_zone_nesting. Если parent == child —
расширяем parent outward для создания иерархии.
