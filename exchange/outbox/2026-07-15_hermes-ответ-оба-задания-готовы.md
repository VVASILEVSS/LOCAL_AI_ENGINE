## 1. Авто-режим — только сигналы в TG ✅

Реализовано в 3 файлах:

### `core/scheduler.py:465-469`
```python
# AUTO_SIGNAL_ONLY: в авто-цикле отправляем только при подтверждённом сигнале
if AUTO_SIGNAL_ONLY:
    send_to_tg = status in ACTIONABLE_SIGNALS
else:
    send_to_tg = True
```
- `AUTO_SIGNAL_ONLY` — глобальный флаг, читается из `get_setting("auto_mode", False)`
- `ACTIONABLE_SIGNALS = ("aggressive_breakout", "retest", "reversal", "false_breakout")`
- Ошибки анализа (`parsed.get("error")`) — ВСЕГДА отправляются (строка 457-461, вне блока `send_to_tg`)
- `save_signal_log` — ВСЕГДА записывается, независимо от режима

### `core/handlers.py:462-468`
```python
@router.message(Command("auto"))
async def cmd_auto(message: types.Message) -> None:
    """Тогл авто-режима: только сигналы в TG."""
    state = not get_setting("auto_mode", False)
    set_setting("auto_mode", state)
    status_text = "🔇 ON (только сигналы)" if state else "📢 OFF (все анализы)"
    await message.answer(f"Авто-режим: {status_text}")
```
