## Заметка по venv

Для запуска нужен изолированный venv (Hermes PYTHONPATH конфликтует с LOCAL_AI_ENGINE deps). Команда:

```bash
cd LOCAL_AI_ENGINE
source .venv/Scripts/activate
PYTHONPATH="" python main.py  # или скрипт прогноза
```

`start_bot.sh` от Super Z создаёт venv и активирует его, но не очищает PYTHONPATH — на Windows с Hermes это может вызвать конфликты (как было у меня).
