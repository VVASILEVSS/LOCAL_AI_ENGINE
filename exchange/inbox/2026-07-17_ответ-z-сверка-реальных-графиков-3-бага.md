## БАГ 1 (критический): D1/4H зоны гигантские — старые пивоты из 2025 года

### Доказательство

OHLCV кэш BTC D1 = 500 свечей (март 2025 → июль 2026). BTC был на **126199** в мае 2025 (строка 218 кэша). Сейчас BTC = **63180**.

**Bot TG вывод:** D1 zone = [57758.60 - **97932.10**]
**Мой тест (новый код):** D1 zone = [57800.19 - **126199.63**]

**Оба WRONG.** Реальный D1 range за последние 30 свечей: [57800 - 67292].

### Root cause: `_STRUCT_WINDOW = None` для D1 и 4H

```python
_STRUCT_WINDOW = {"5m": 50, "15m": 50, "1h": 80, "4h": None, "1d": None}
```

None = **все 500 свечей**. ZigZag находит пивоты с мая 2025 (126k для BTC, 4956 для ETH). BOS detection находит старый BOS → curr_struct включает свинги из 2025 года → zone = огромная.

**Фикс:** задать window для D1 и 4H.

```python
_STRUCT_WINDOW = {"5m": 50, "15m": 50, "1h": 80, "4h": 150, "1d": 100}
```

D1=100 свечей (~3 месяца), 4H=150 (~25 дней). Достаточно для структуры, отсекает древние экстремумы.
