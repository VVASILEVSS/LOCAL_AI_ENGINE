## §3. ТИПОВЫЕ ПРОВОДКИ

### Зарплата
```
Начисление:    Дт 7210 / Кт 3350 = 72 250
Налоги (ежем): Дт 7210 / Кт (3150+3211+3212+3220) = 30 325
Оплата налогов:Дт (3150+3211+3212+3220) / Кт 1030 (или 1040) = 30 325
```

### Kaspi Pay (1040)
```
Оплата от клиента: Дт 1040 / Кт (клиент) = сумма
Оплата налогов:    Дт 3xxx / Кт 1040 = сумма
Вывод на Kaspi Gold:Дт 3350 / Кт 1040 = сумма
```

### Авансы (3300 — кредиторка)
```
Выдача аванса:    Дт 3300 / Кт 1030 (или 1040) = сумма
Поступление товара:Дт 1300 / Кт 3300 = сумма
Возврат аванса:   Дт 1030 (или 1040) / Кт 3300 = сумма
```

### Подотчёт директора (1250)
```
Выдача аванса:     Дт 1250 / Кт 1030 (или 1040) = сумма
Расходы по отчёту: Дт 7210 (или 7110) / Кт 1250 = сумма
Возврат остатка:   Дт 1030 (или 1040) / Кт 1250 = сумма
```

### ForteBank карта (1031)
```
Перевод на карту: Дт 1031 / Кт 1030 = сумма
Покупки:          Дт 1300 / Кт 1031 = сумма
```

### Прибыль прошлых лет
```
Дт 000 / Кт 5620 = сумма (НЕ 6000, НЕ 5010, НЕ 5610!)
```

## §5. ИЗВЕСТНЫЕ ОШИБКИ — НЕ ПОВТОРЯТЬ!

| # | Ошибка | Почему неверно | Правильно |
|---|--------|---------------|-----------|
| 1 | Субсчёт 1030.02 через точку | Не работает в 1С | Отдельный номер (1031) |
| 2 | ОПВ и ОПВР на разных счетах | В 1С КЗ они вместе | ВМЕСТЕ на 3220 |
| 3 | СО и соц. налог на одном счёте | Разные вещи | СО→3211, соц.налог→3150 |
| 4 | Считать С-1 пропорционально | Соц.налог от дохода, не плоская ставка | ТОЛЬКО из справки |
| 5 | С-1 за янв-июнь (6 мес) | В справке есть Декабрь 2025 | Дек 2025–Июнь 2026 = 7 мес |
| 6 | ИПН каждый месяц | ОУР = раз в год | НЕ ТРОГАТЬ, копится в Кт 3120 |
| 7 | Прибыль на 6000/5010/5610 | 6000=доход (налог уплачен), 5010=резерв, 5610=текущий год | Дт 000/Кт 5620 |
| 8 | Дт 000Н/Кт 5610Н | У 56xx нет версий «Н» | Обычные счета, 000Н автоматически |

## §6. ЧЕК-ЛИСТ ОСВ

После каждой группы проводок:
- [ ] Баланс сходится: Итого Дт = Итого Кт
- [ ] 000 = 0
- [ ] 000Н = 0
- [ ] 3100 = 0 (если налоги уплачены)
- [ ] 3200 = 0 (если нет переплат)
- [ ] 1300 Дт = остаток товара на складе
- [ ] 5600 Кт = нераспределённая прибыль

## §10. ПАПКА questions/ — «КУРИЛКА»

**Назначение**: общее пространство для общения по 1С — вопросы, проблемы, решения, обсуждения.

Правила:
- Каждый вопрос/проблема = отдельный файл: `questions/Q_NNN_краткое_описание.md`
- Формат файла:
  1. **Описание** — суть вопроса или проблемы
  2. **Обсуждение** — что пробовали, варианты, рассуждения
