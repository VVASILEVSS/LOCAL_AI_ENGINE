## Сводка за 02.06.2026 — LOCAL_AI_ENGINE, A/D Divergence Module

---

### 1. divergence_context.py v2.4 — удаление лимитов

**Проблема:** `get_multi_symbol_context()` имела `max_symbols=3` — блокировало работу TG бота с произвольными тикерами Binance.

**Решение:**
- Убран `max_symbols` полностью
- Вместо него — контроль через размер промпта:
  - `max_per_tf` = 5 (сигналов на символ×TF)
  - `max_total_signals` = 30 (общий лимит)
  - `max_prompt_chars` = 4000 (жёсткий cap, авто-сокращение)
- Добавлен **TTL-кэш** (30 мин) — JSON не перечитывается каждый запрос
- Добавлены `discover_available_symbols()` и `discover_available_tfs()` для TG бота

### 2. ollama_client.py — multi-symbol интеграция

**Изменения:**
- Импорт: `get_multi_context` + `get_multi_symbol_context`
- `analyze_multi_images()` теперь читает из `prev_analysis`:
  - `divergence_symbols` (str → [str], backward compatible)
  - Если 1 символ → `get_multi_context()`, если N → `get_multi_symbol_context()`
  - Все параметры настраиваемые: `divergence_max_per_tf`, `divergence_max_total_signals`, `divergence_max_prompt_chars`

**Формат multi-symbol вывода:**
```
A/D Divergence (3 symbols, 1h, 4h, lookback 168h):
[BTCUSDT]
  4H (3, 2 reg, 1 hid):
    1. BULL (regular) | Str: 8.2/10 | Move: +1.8% | Bias: +4.3 | Vol: 2.1x
[ETHUSDT]
  1H (2, ...):
    ...
```
